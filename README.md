## 📋 О проекте

Данный репозиторий содержит мои решения практических задач и кейсов, предлагаемых в программе обучения. Цель курса — освоение полного цикла работы с данными: от сбора и очистки до построения моделей машинного обучения и визуализации результатов.
Все работы выполнены на языке Python с использованием стандартных библиотек для анализа данных в среде Jupyter Notebook.

## 🗂️ Структура репозитория

Репозиторий разбит на модули в соответствии с учебным планом:

TBank-Analytics/

├── 📁 Data processing using SQL and Python/

│   └── *.ipynb          # Загрузка данных, очистка, агрегация, работа с NULL, фильтрация и преобразование данных средствами Python/Pandas

│

├── 📁 Probability theory and mathematical statistics/

│   └── *.ipynb          # Описательная статистика, проверка гипотез, A/B-тесты, доверительные интервалы


│                        
├── 📁 Machine learning/

│   └── *.ipynb          # Построение моделей классификации/регрессии, feature engineering, оценка качества (Accuracy, ROC-AUC)

│

├── 📁 Visualisation/

│   └── *.ipynb          # Построение графиков и диаграмм: гистограммы, scatter plot, box plot, heatmap корреляций

## В процессе выполнения заданий были отработаны следующие навыки:

✅ Обработка данных: загрузка, очистка, работа с пропусками, агрегация и фильтрация в Pandas

✅ Статистический анализ: описательная статистика, проверка гипотез, доверительные интервалы

✅ A/B-тестирование: проектирование экспериментов, оценка статистической значимости результатов

✅ Машинное обучение: построение моделей классификации и регрессии, feature engineering

✅ Оценка моделей: метрики Accuracy, Precision, Recall, ROC-AUC, кросс-валидация

✅ Визуализация: гистограммы, scatter plot, box plot, heatmap корреляций
